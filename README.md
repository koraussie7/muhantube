# MuhanTube 🎬

MuhanTube — Decentralized video platform powered by Vidstack + P2P + Minima

## Monorepo Structure

apps/web/              Vue 3 + Vite web app
packages/player/       Vidstack + SwarmCloud P2P player
legacy/                DTube legacy code

## Quick Start

pnpm install
pnpm dev
pnpm build

## Stack

Player: Vidstack v1.15+ / HLS.js
P2P: SwarmCloud
Frontend: Vue 3 + Vite + Tailwind
State: Pinia
Blockchain: Minima MDS
Content API: Avalon
